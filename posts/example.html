发帖完全指南：如何写出一篇完美的文章

2025-10-24 教程 网站开发

欢迎使用全新的 Maple2085 博客排版系统！这篇文章不仅是一篇示范，也是一份详细的使用教程。本系统采用了顶级科技公司（如 Apple、Linear）推崇的“玻璃态（Glassmorphism）”排版语言。

1. 如何新建一篇文章？

发帖的流程非常简单，完全不需要数据库，只需要操作本地文件：

  1. 在 posts/ 文件夹中，复制这个 example.html 文件。
  2. 将它重命名为一个有意义的短英文名字，比如：my-first-post.html。
  3. 用编辑器打开那个文件，修改顶部的<title>、大标题 <h1>，然后在这个 <div class="prose"> 里面畅所欲言。
提示： 不要直接在 index.html 里写文章！posts.html 会自动读取 JSON，你要做的只是写好独立的 HTML 文件。

2. 如何在首页列表里显示我的文章？

你写好了 my-first-post.html 后，列表页还不知道它的存在。你需要更新 posts.json 索引文件。

打开 posts/posts.json，在里面添加一个新的 JSON 对象：

{
  "id": "my-first-post",
  "title": "这是我的第一篇文章",
  "tags": ["日常", "碎碎念"],
  "date": "2025-10-25",
  "summary": "这是显示在卡片上的简短摘要，最好控制在两句话以内。",
  "cover": "covers/my-cover.jpg" 
}
  • id：最重要！它必须和你新建的文件名一模一样（不带 .html）。比如你的文件名是 my-first-post.html，这里就填 my-first-post。
  • cover：如果不填，卡片会显示一个默认的图片图标。
  • tags：可以填多个，列表页会自动根据这些标签生成“筛选按钮”。

3. Markdown 常见排版演示

在这个系统里，你不需要配置复杂的 Markdown 转换器，直接写 HTML 标签就能获得顶级的视觉体验。

3.1 强调与链接

想要重点突出，可以用 <strong> 标签加粗文字。如果涉及专业词汇，可以使用行内代码标签，比如 <code>JavaScript</code> 会显示为 JavaScript。

如果想要插入超链接，直接写 枫叶的 GitHub 主页，我已经为你写好了悬浮时的背景高亮特效。

3.2 引用块 (Blockquote)

如果你想引用别人的话，或者做个显眼的总结，可以用 blockquote：

“代码写出来是给人看的，只是恰好机器也能运行。”
—— 某位著名的程序员

直接使用 img 标签即可，图片会自动加上圆角和柔和的阴影，并且不会超出屏幕宽度：
