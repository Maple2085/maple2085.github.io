核心功能

Txt Splitter 是为了解决网络小说、超大日志文件无法直接阅读而诞生的轻量化工具。它具备以下优势：

  • 按大小分割：精准控制每个切片的大小（例如每份 500KB）。
  • 智能章节识别：内置起点、晋江等常见小说网站的章节特征识别。
  • 正则表达式提取：为高级用户提供自定义的分割规则。
  • 纯前端处理：不消耗服务器资源，断网也能用，文件绝对安全。

快速部署

如果您想将本项目部署到自己的服务器上，只需执行以下命令：

bash — txt-splitter
# 1. 克隆项目到本地
git clone https://github.com/Maple2085/txt-splitter.git

# 2. 进入目录并安装依赖
cd txt-splitter
npm install

# 3. 启动本地开发服务器
npm run dev
